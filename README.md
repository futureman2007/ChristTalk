## ‚ú® Features (Planned)

- **Account-based messaging** (no phone number required)
- **Private & secure chats** (future goal: end-to-end encryption)
- **Radar Mode**
  - Discover other users in your physical vicinity
  - Bluetooth / Wi-Fi / local peer-to-peer discovery
  - Chat locally without a central server
- **Online server mode** for global communication
- **Community-friendly, open-source environment**
- **Modular Kotlin codebase**
- **MIT licensed**

## üî≠ Vision

ChristTalk combines modern chat technologies with a Christian community focus:

- Provide a safe, respectful, and faith-oriented platform
- Enable spontaneous, local communication between believers
- Allow worldwide communication when desired
- Ensure full privacy by minimizing required personal information
- Provide transparency through open-source development

## üõ† Technical Goals (Early Stage)

- Optional future server backend (Node.js, Go, or Kotlin backend)
- Clean architecture + testable core modules
- Modular design for community contributions
